### OpenRouter — $10 один раз (для доступа к бесплатным лимитам)

**Сайт**: https://openrouter.ai  
**Цены**: https://openrouter.ai/pricing  

#### Что это
Агрегатор API для доступа к 290+ моделям через один ключ.

#### Цены
- **Free**: $0 (50 запросов/день на :free модели)
- **С $10**: 1000 запросов/день на :free модели
- **Pay-as-you-go**: платные модели по ценам провайдеров

**Важно**: $10 покупаются один раз, не сгорают, остаются на балансе.

#### Как открыть 1000 запросов/день

**Официальная документация**: https://openrouter.ai/docs/faq

Из документации:
> "If you have purchased at least $10 credits, your free model rate limit will be 1000 requests per day. Otherwise, you will be rate limited to 50 free model API requests per day."

**Инструкция**:
```
1. Зарегистрироваться: https://openrouter.ai
2. Перейти в Credits: https://openrouter.ai/credits
3. Купить минимум $10 кредитов
4. Лимиты автоматически повысятся до 1000/день
```

**Проверка лимитов**:
```bash
# API запрос
curl https://openrouter.ai/api/v1/key \
  -H "Authorization: Bearer YOUR_API_KEY"

# Ответ покажет:
# - limit: текущий дневной лимит
# - usage: использовано сегодня
# - is_free_tier: true/false
```

**Проверка в веб-интерфейсе**:
```
1. Открыть https://openrouter.ai/settings/limits
2. Увидите: "Daily free model limit: 1000 requests"
```

#### Лимиты
**Без покупки** ($0):
- 50 запросов/день
- 20 запросов/минуту

**С покупкой $10+**:
- 1000 запросов/день
- 20 запросов/минуту

#### Бесплатные модели для кода
- `qwen/qwen3-coder:free` — лучшая для кода
- `deepseek/deepseek-r1:free` — для reasoning
- `meta-llama/llama-3.3-70b:free`

#### Плюсы
- $10 один раз (не подписка)
- 1000 free запросов/день
- Доступ к 290+ моделям
- Кредиты можно использовать для платных моделей

#### Минусы
- Нужно работать с API
- Free модели медленнее в пик нагрузки
- Не для новичков
